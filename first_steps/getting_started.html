Getting Started¶

Adding DroidParts to your own app is easy.

  1. Clone the sources from https://github.com/yanchenko/droidparts.git. The suggested branch is master as it contains the latest release.
  2. Import the existing Eclipse projects.
  3. ActionBarSherlock is required at compile time. To avoid this, use droidparts.jar. (TODO)
  4. Add dependencies to your project.
  5. Configure AdnroidManifest.xml
  6. Subclass Application.
  7. Subclass AbstractDependencyProvider.
  8. Proguard

AndroidManifest.xml Options¶

Dependency Injection & Logging have options configurable via AndroidManifest.xml. The first one must be configured.
